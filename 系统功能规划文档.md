# 代账公司管理系统 — 功能规划文档

> 版本：v1.0 | 日期：2026/05/28

---

## 一、系统概述

本系统面向代理记账/财务服务公司，提供客户管理、员工管理、薪酬核算、收费管理、工作日志等一体化功能，实现公司内部运营的数字化、规范化管理。

**技术架构：**
- 后端：Python Django
- 数据库：MySQL
- 前端：Bootstrap + jQuery（AdminLTE 管理后台模板）
- 部署：B/S 架构，浏览器访问

---

## 二、系统角色

| 角色 | 说明 | 权限范围 |
|------|------|---------|
| 管理员 | 系统最高权限 | 管理所有功能、系统设置、用户管理、数据监控、全局统计 |
| 主管（主管会计） | 团队管理者 | 查看所有员工及客户信息、查看员工工作日志、管理薪酬核算、分配客户 |
| 会计（普通员工） | 业务执行者 | 管理自己对接的客户、填写工作日志、查看自己的薪酬、催收对接客户费用 |

**权限隔离规则：**
- 客户信息：仅对接该客户的会计 + 主管及以上可见
- 工作日志：仅本人 + 主管及以上可查看
- 薪酬信息：仅本人可查看自己的，主管/管理员可查看全部
- 客户无需登录系统，仅作为管理对象

## 三、核心功能模块

### 3.1 客户（企业）管理

**功能描述：** 管理代账公司服务的所有企业客户信息。

| 功能 | 说明 |
|------|------|
| 客户信息录入 | 企业名称、企业性质、经营范围、法人代表、联系人、联系方式（手机/座机）、地址、客户编号 |
| 客户列表 | 分页展示、支持按企业名称/联系人搜索、按状态筛选 |
| 客户编辑/删除 | 修改客户信息、停用/启用客户 |
| 缴费信息管理 | 缴费开始日期、缴费周期（按月/季/年）、缴费金额 |
| 缴费记录 | 记录每笔缴费的应收日期、实收日期、金额、状态（已缴/欠费） |
| 催收提醒 | 根据缴费周期自动生成到期提醒，通知对接会计进行催收 |
| 客户统计 | 客户总数、活跃客户数、欠费客户数等 |

### 3.2 员工（会计）管理

**功能描述：** 管理公司内部会计/财务人员信息。

| 功能 | 说明 |
|------|------|
| 员工信息录入 | 姓名、性别、年龄、学历、专业、联系方式（手机/邮箱）、入职日期 |
| 证书管理 | 会计证、中级会计师证等（影响基本工资计算） |
| 工龄记录 | 自动计算或手动录入（影响基本工资计算） |
| 员工列表 | 分页展示、搜索、按状态/角色筛选 |
| 员工编辑/删除 | 修改信息、离职处理 |
| 客户分配 | 将客户指派给对应会计，支持一个会计对接多个客户 |
| 工作负载查看 | 查看每个会计当前对接的客户数量和工作量 |

### 3.3 薪酬核算

**功能描述：** 自动计算员工月度薪酬，支持导出工资表。

| 项目 | 计算规则 |
|------|---------|
| 基本工资 | 固定部分，依据学历、证书数量、工龄动态调整（学历越高/证书越多/工龄越长，基本工资越高） |
| 绩效工资 | 按工作量计算：对接客户数 × 单价（简单客户100元，复杂客户200元）+ 服务内容单价（报税/开票等）+ 开票数量提成 |
| 总薪酬 | 基本工资 + 绩效工资 |

**核心功能：**
- 月度薪酬自动核算
- 员工工资明细查看
- 月度工资表导出（全员）
- 公司月收入统计（客户缴费总和）
- 公司月支出统计（员工薪酬总和）
- 公司月收益计算（收入 - 支出）

### 3.4 工作日志

**功能描述：** 员工每日记录工作内容，主管可查阅。

| 功能 | 说明 |
|------|------|
| 日志填写 | 员工每天记录工作内容（做什么账、开什么票、报税等） |
| 日志查看 | 本人查看自己的历史日志 |
| 主管查阅 | 主管可查看所有下属的工作日志 |
| 日志筛选 | 按日期范围、员工、工作类型筛选 |

### 3.5 收费管理

**功能描述：** 管理客户服务收费标准和缴费流程。

| 功能 | 说明 |
|------|------|
| 收费标准设置 | 按客户类型、服务项目、周期灵活设置收费标准和价格 |
| 账单生成 | 根据收费标准和服务周期，自动生成客户应缴账单 |
| 催缴通知 | 系统自动检测即将到期/已逾期费用，提醒对接会计催收 |
| 收费记录 | 详细记录每笔费用的应收、实收、欠款、备注 |

## 四、系统功能流程

```
┌─────────────────────────────────────────────────────────┐
│                    管理员/主管                            │
│         添加客户信息 ──── 添加员工信息                     │
└────────────────────────┬────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────┐
│              建立会计 ↔ 客户对接关系（一对多）              │
└────────────────────────┬────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────┐
│          会计为客户提供服务（报税、开票、做账）             │
│              ↓ 每日填写工作日志                           │
└────────────────────────┬────────────────────────────────┘
                         │
              ┌──────────┴──────────┐
              ▼                     ▼
┌──────────────────────┐  ┌──────────────────────┐
│    薪酬自动核算       │  │    收费催收管理       │
│ 基本工资 + 绩效工资   │  │ 按周期提醒 → 会计催收 │
│ → 月度工资表导出      │  │ → 记录缴费状态       │
└──────────────────────┘  └──────────────────────┘
              │                     │
              └──────────┬──────────┘
                         ▼
┌─────────────────────────────────────────────────────────┐
│                  公司经营报表                             │
│        月收入（客户缴费） - 月支出（员工薪酬） = 收益      │
└─────────────────────────────────────────────────────────┘
```

## 五、数据库核心表设计

| 表名 | 主要字段 | 说明 |
|------|---------|------|
| User（用户表） | 账号、密码、姓名、性别、学历、专业、手机、邮箱、证书、工龄、角色、状态 | 系统所有用户（管理员/主管/会计） |
| Client（客户表） | 客户编号、企业名称、企业性质、经营范围、法人、联系人、电话、地址、缴费周期、缴费金额、状态 | 企业客户信息 |
| AccountantClient（会计-客户关联表） | 会计ID、客户ID、分配日期 | 记录对接关系 |
| Fee（收费记录表） | 客户ID、应收日期、应收金额、实收日期、实收金额、状态、备注 | 每笔缴费记录 |
| Salary（薪酬表） | 员工ID、月份、基本工资、绩效工资、总薪酬 | 月度薪酬记录 |
| WorkLog（工作日志表） | 员工ID、日期、工作类型、工作内容、关联客户ID | 每日工作记录 |

## 六、页面规划

### 后台管理页面

| 页面 | 功能 | 可见角色 |
|------|------|---------|
| 登录页 | 账号密码登录 | 所有人 |
| 仪表盘 | 数据概览（客户数、员工数、本月收入/支出/收益） | 管理员、主管 |
| 客户列表 | 客户增删改查、搜索筛选 | 管理员、主管、会计（仅看自己的） |
| 客户详情 | 客户完整信息 + 缴费记录 | 管理员、主管、对接会计 |
| 员工列表 | 员工增删改查 | 管理员、主管 |
| 员工详情 | 员工信息 + 对接客户 + 薪酬 | 管理员、主管 |
| 客户分配 | 将客户指派给会计 | 管理员、主管 |
| 收费管理 | 收费标准设置、缴费记录、催收列表 | 管理员、主管 |
| 薪酬管理 | 月度薪酬核算、工资表导出 | 管理员、主管 |
| 工作日志 | 填写/查看工作日志 | 会计（填写+查看自己）、主管（查看全部） |
| 我的薪酬 | 查看个人薪酬明细 | 会计 |
| 催收提醒 | 待催收客户列表 | 会计（自己的客户）、主管（全部） |
| 经营报表 | 月度收入/支出/收益统计 | 管理员、主管 |

## 七、非功能需求

| 项目 | 说明 |
|------|------|
| 安全性 | 密码加密存储、登录会话管理、防止越权访问 |
| 可扩展性 | 模块化设计，后续可扩展 API 接口、移动端适配、AI 智能分析等 |
| 易用性 | 界面简洁清晰，操作流程符合业务习惯 |
| 数据备份 | 支持数据库定期备份 |
